# Excel Sales Data Analysis

A hands-on Excel project analyzing 50 sales transactions to extract business insights using core Excel functions.

## What This Project Does
- Calculates revenue per transaction (Units Sold × Price)
- Flags high vs low performing sales using IF logic
- Aggregates revenue by region using SUMIF
- Counts orders per salesperson using COUNTIF

## Excel Functions Used
| Function | Purpose |
|----------|---------|
| Basic Formula (E*F) | Calculate revenue per row |
| SUM | Total revenue across all transactions |
| IF | Classify performance as High/Low |
| SUMIF | Region-wise revenue breakdown |
| COUNTIF | Count orders per salesperson |

## Dataset
50 rows of sales data including Date, Product, Category, Region, Units Sold, Price, and Salesperson.

## Key Insight Example
Chennai region generated ₹6,39,050 in total revenue — identified instantly using SUMIF instead of manual filtering.

## Lookup Functions Added
- VLOOKUP for product price lookup
- XLOOKUP as the modern alternative (no column counting needed)
- IFERROR wrapping to handle missing values gracefully
